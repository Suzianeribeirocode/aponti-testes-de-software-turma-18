# 📌 Objetivo

Este projeto foi desenvolvido para praticar a linguagem **Markdown**, aprendendo a criar um **README profissional**, organizado e visualmente agradável.

Neste projeto foram utilizados:

- **Títulos**
- **Negrito**
- *Itálico*
- Listas
- Tabelas
- Emojis 😄
- Links
- Imagens
- PDFs
- Blocos de código
- Badges
- Muito mais...
